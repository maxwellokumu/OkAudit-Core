# data-privacy-playbook

Step-by-step data privacy audit playbook covering 5 phases: data discovery and classification, legal basis and consent review, data subject rights verification, PIA/DPIA execution, and reporting. Links directly to all data-privacy skills.

## Requirements

```
python-dotenv
```

## Usage

### Run full playbook
```bash
python main.py
```

### Run a specific step
```bash
python main.py --step 4
```

## Steps

| Step | Title | Linked Skills |
|------|-------|--------------|
| 1 | Data Discovery & Classification | data-inventory-mapper |
| 2 | Legal Basis & Consent Review | consent-checker |
| 3 | Data Subject Rights Verification | — |
| 4 | PIA / DPIA Execution | pia-generator |
| 5 | Reporting & Remediation | All data-privacy skills |
