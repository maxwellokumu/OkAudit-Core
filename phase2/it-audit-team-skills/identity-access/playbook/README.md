# IAM Audit Playbook

Step-by-step methodology for Identity & Access Management audits, covering all
six phases from user inventory through final reporting.

---

## Usage

```bash
# Full playbook
python main.py

# Specific step
python main.py --step 2
python main.py --step 5
```

## Steps

| Step | Title |
|------|-------|
| 1 | User & Account Inventory |
| 2 | IAM Policy Review |
| 3 | MFA Verification |
| 4 | Privileged Account Monitoring |
| 5 | Segregation of Duties Analysis |
| 6 | Reporting & Remediation Tracking |
