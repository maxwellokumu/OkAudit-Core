# hardware-physical/playbook

A step-by-step audit playbook for hardware and physical security reviews. Covers asset inventory reconciliation, physical access log analysis, firmware vulnerability assessment, environmental controls, and remediation reporting.

## Requirements

- Python 3.8+
- No external dependencies

## Usage

```bash
# Full playbook
python main.py

# Single step
python main.py --step 3
```

### Steps

| Step | Title |
|------|-------|
| 1 | Asset Inventory & Reconciliation |
| 2 | Physical Access Review |
| 3 | Firmware & Patch Status |
| 4 | Environmental Controls Assessment |
| 5 | Reporting & Remediation |
