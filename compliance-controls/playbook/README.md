# Compliance & Controls Audit Playbook

Step-by-step methodology for compliance and controls audits, from framework
selection through to final reporting.

---

## Usage

```bash
python main.py               # Full playbook
python main.py --step 3      # Control testing step only
```

## Steps

| Step | Title |
|------|-------|
| 1 | Control Framework Selection |
| 2 | Evidence Request Planning |
| 3 | Control Testing Procedures |
| 4 | Exceptions and Gap Management |
| 5 | Compliance Reporting |
